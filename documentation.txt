This is an implentation of a symbol table represented on a hash table that uses separate chaining as a collision resolution method.

Operations:

*init(self):
	description: Creates a new empty Symbol Table


*hash(key):
	description:  Generates a hash for a given key
	pre: key - string
	post: index - integer

*insert(key, value):
	description:  Inserts a key,value pair to the smyboltable
	pre: key - string, value - integer
	
*find(key):
	description: Finds a data value based on key
	pre: key - string
	post: values - list of values, index - integer


*remove(key):
	description:  Remove node stored at key
	pre: key - string
	post: node - Node
